# square-game
---
## Install:
run on your favorite shell:
```
pip install git+https://github.com/tristanML/square-game.git
```
Issues with `pygame`:
+ on MacOS
```bash
brew install sdl sdl_image sdl_mixer sdl_ttf portmidi
```
## Play:
```
square-game-play
```
